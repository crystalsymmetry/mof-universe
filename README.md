This is the home of the MOF Universe 3.0. The version 3.0 is an extension of the existing MOF 2.0 Universe, hosted on a wordpress site:
https://crystalsymmetry.wordpress.com/mof-universe/
There you can also find a Google spread sheet that contains more data on all the listed groups working on MOFs/COFs/ZIFs etc.
Beyond the visual appealing rotating MOF globe, the extended version 3.0 includes now a search functionality. You can also filter entries by 
clicking on one of the categegories (hide/show toggle) in the legend.
